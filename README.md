# retail-sales-data-analysis
End-to-end retail sales analysis using Python, SQL, and Power BI
Project Overview

This project demonstrates an end-to-end data analysis workflow using Python, SQL, and Power BI.

The goal of the project is to analyze retail sales data to understand product performance, customer behavior, and revenue trends.

The analysis includes:

.Data cleaning and preparation

.Exploratory data analysis

.SQL business queries

.Interactive Power BI dashboard

Project Workflow

  1️⃣ Data Preparation (Python)

      Created retail datasets for customers, products, and orders

      Cleaned and validated data

      Converted date columns

      Checked for duplicates and missing values

      Generated new features such as revenue and month columns

 2️⃣ Exploratory Data Analysis

     Using Python, the following insights were explored:

         Revenue by product

         Revenue by city
 
         Revenue by category

         Monthly revenue trends

         Visualizations were created using Matplotlib.

 3️⃣ SQL Business Analysis

     Data was loaded into SQLite and analyzed using SQL queries including:

         Total revenue per product

         Top spending customers

         Revenue by city

         Customer spending ranking

         Month-over-month revenue growth

 4️⃣ Power BI Dashboard

     An interactive dashboard was created to visualize business insights.

    Dashboard features include:

         Total Revenue KPI

         Total Orders KPI

         Total Customers KPI

         Revenue by Product

         Revenue by City

         Revenue trend over time

         Interactive city slicer
